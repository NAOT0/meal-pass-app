# ミールパス計算アプリ (Meal Pass App)

大学生協のミールパスをスマートに活用するための計算・管理アプリです。

## 🚀 主な機能
- **予算計算**: 1日の限度額に合わせて最適な商品の組み合わせを提案
- **バーコードスキャン**: 商品のバーコードから即座に価格と情報を取得
- **仕分けゲーム**: ユーザー同士で協力して商品データを精査
- **マルチプラットフォーム**: iOS / Android / Web (PWA) 対応

## 🛠 技術スタック
- Expo SDK 54 / Expo Router v3
- React Native / NativeWind (Tailwind CSS)
- Supabase (Backend / Database)
- Vercel (Web Hosting)

## 📖 開発者向けドキュメント

## 開発の始め方
```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npx expo start
```
